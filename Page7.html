Course Objectives

  • Understand the fundamentals of image processing and computer vision
  • Learn about image formation, enhancement, and transformations
  • Study feature detection, matching, and camera geometry
  • Explore depth estimation and structure from motion
  • Gain knowledge of deep learning techniques in computer vision
  • Understand various applications of computer vision

Course Outcome

  • Apply image processing techniques for enhancement and filtering
  • Implement feature detection and matching algorithms
  • Understand and apply camera geometry concepts
  • Perform depth estimation and structure from motion
  • Utilize deep learning models for computer vision tasks
  • Develop solutions for real-world computer vision applications

Modules

Module 1

Introduction, Image Formation, Image Processing, Feature Detection

Module 2

Feature Detection and Matching, Camera Geometry, Depth Estimation, Structure from Motion, Object Tracking

Module 3

Deep Learning: Neural Networks, CNNs, RNNs, Transformers, Generative Models
